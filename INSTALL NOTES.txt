1. Install Python 3.x (from https://www.python.org/downloads/)
	-Include Python in the path during installation.

2. Install ImageMagick 7.x (from https://imagemagick.org/script/download.php)
	-Include the legacy command line option during installation.
	
3. Install Pillow.
	-From a terminal install Pillow with pip:
	pip install Pillow
